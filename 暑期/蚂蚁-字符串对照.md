```
输入

12
Ab01!?. +-*/
aaaAaaaaaaaa   
输出

Aa9765________
说明

在这个样例中，输入的字符串为 “Ab01!?.  +_*/” （注意第八个字符为空格）
```

```python
n = int(input())
s = input()
t = input()

res = ""
for i in range(n):
    if'A' <= s[i] <= 'Z':
        res += t[i].upper()
    elif'a' <= s[i] <= 'z':
        res += t[i].lower()
    elif'0' <= s[i] <= '9':
        res += str(ord(t[i]))
    else:
        res += "_"

print(res)
```
